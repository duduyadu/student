1. 시스템 소개
================================================================================

AJU E&J 학생관리 시스템은 베트남 학생의 한국 유학·취업을 지원하는
유학원 AJU E&J를 위해 만들어진 웹 기반 관리 프로그램입니다.

주요 기능:
  - 학생 정보 등록 및 관리
  - 비자 만료일 추적 및 자동 이메일 알림
  - 유학원별 학생 관리
  - 학생 본인 정보 확인 포털
  - 통계 및 리포트


================================================================================
2. 사용자 종류
================================================================================

이 시스템에는 세 종류의 사용자가 있습니다.

  ┌─────────────┬──────────────────────────────────────────────────┐
  │  사용자     │  할 수 있는 일                                   │
  ├─────────────┼──────────────────────────────────────────────────┤
  │  관리자     │  모든 학생·유학원 관리, 계정 생성, 통계 확인     │
  │  (마스터)   │                                                  │
  ├─────────────┼──────────────────────────────────────────────────┤
  │  유학원     │  담당 학생 목록 확인 및 기본 관리                │
  ├─────────────┼──────────────────────────────────────────────────┤
  │  학생       │  본인 정보 확인만 가능 (수정 불가)               │
  └─────────────┴──────────────────────────────────────────────────┘


================================================================================
3. 로그인 방법
================================================================================

1) 인터넷 브라우저(크롬 권장)에서 아래 주소를 입력합니다.
   https://aju-ej.vercel.app/login

2) 이메일과 비밀번호를 입력하고 [로그인] 버튼을 클릭합니다.

3) 역할에 따라 자동으로 해당 화면으로 이동합니다.
   - 관리자 → 대시보드
   - 유학원 → 학생 목록
   - 학생   → 포털 (본인 정보)

※ 비밀번호를 잊었을 경우:
   로그인 화면에서 [비밀번호 찾기]를 클릭하면 이메일로 재설정 링크가 발송됩니다.


================================================================================
4. 관리자 사용법
================================================================================

--------------------------------------------------------------------------------
4-1. 대시보드
--------------------------------------------------------------------------------

로그인 후 첫 화면입니다. 다음 정보를 한눈에 볼 수 있습니다.

  - 전체 학생 수
  - 30일 이내 비자 만료 학생 수
  - 상태별 학생 수 (유학전 / 어학연수 / 대학교 / 취업)
  - 상태별 비율 바 차트

상단 메뉴에서 다른 페이지로 이동할 수 있습니다.
  [대시보드] [학생 관리] [리포트] [유학원 관리]


--------------------------------------------------------------------------------
4-2. 학생 목록
--------------------------------------------------------------------------------

메뉴에서 [학생 관리]를 클릭하면 등록된 모든 학생을 볼 수 있습니다.

  ▶ 검색: 이름(한국어/베트남어) 또는 여권번호로 검색
  ▶ 필터: 상태별(유학전/어학연수/대학교/취업), 유학원별 필터
  ▶ 정렬: 비자 만료일 기준 정렬
  ▶ Excel 내보내기: [Excel] 버튼을 클릭하면 전체 학생 목록을 파일로 저장

학생 카드를 클릭하면 상세 정보를 볼 수 있습니다.


--------------------------------------------------------------------------------
4-3. 학생 등록
--------------------------------------------------------------------------------

[+ 학생 등록] 버튼을 클릭합니다.

입력 항목:
  [기본 정보]
  - 한국어 이름 (필수)
  - 베트남어 이름 (필수)
  - 생년월일
  - 연락처 (전화번호)
  - 이메일
  - 현재 상태: 유학전 / 어학연수 / 대학교 / 취업
  - 현재 소속 (상태에 따라 표시됨)
    · 어학연수 선택 시 → 재학 중인 어학원 이름
    · 대학교 선택 시   → 재학 중인 대학교 이름
    · 취업 선택 시     → 재직 중인 회사 이름
  - 담당 유학원 선택

  [비자 정보]
  - 여권번호
  - 비자 종류
  - 비자 만료일 (입력 시 자동 알림 발송)

  [기타]
  - 포털 이메일 (학생이 포털 로그인에 사용할 이메일)
  - 포털 비밀번호 (초기 비밀번호 설정)
  - 비고

입력 완료 후 [저장] 버튼을 클릭합니다.


--------------------------------------------------------------------------------
4-4. 학생 수정 및 상세 보기
--------------------------------------------------------------------------------

학생 목록에서 해당 학생을 클릭하면 상세 페이지로 이동합니다.

탭 구성:
  [기본 정보] - 이름, 연락처, 상태, 소속
  [학업 정보] - 어학원/대학교/회사, 담당 유학원
  [비자 정보] - 여권번호, 비자 종류, 만료일
  [비고]      - 메모 내용

수정 방법:
  우측 상단 [수정] 버튼 클릭 → 내용 변경 → [저장]

비활성화 (탈퇴 처리):
  [비활성화] 버튼을 클릭하면 학생이 목록에서 숨겨집니다.
  (데이터는 삭제되지 않고 보관됩니다)


--------------------------------------------------------------------------------
4-5. 학생 Excel 일괄 등록
--------------------------------------------------------------------------------

여러 명의 학생을 한 번에 등록할 수 있습니다.

1) [학생 관리] → [Excel 일괄 등록] 클릭
2) 샘플 파일을 다운로드하여 형식 확인
3) Excel 또는 CSV 파일에 학생 정보 입력
4) 파일 업로드 후 [등록] 클릭


--------------------------------------------------------------------------------
4-6. 유학원 관리
--------------------------------------------------------------------------------

메뉴에서 [유학원 관리]를 클릭합니다.

  ▶ 유학원 등록:
     [+ 유학원 추가] 버튼 클릭
     - 유학원명(베트남어) 필수 입력
     - 유학원명(한국어), 담당자명, 연락처 선택 입력
     - 계정 이메일/비밀번호 입력 시 유학원 로그인 계정 즉시 생성

  ▶ 계정 추가 (직원 2명 이상인 경우):
     유학원 항목에서 [계정 추가] 버튼 클릭
     - 추가 이메일/비밀번호 입력 → 같은 유학원 권한의 계정 생성

  ▶ 비밀번호 재설정:
     계정이 있는 유학원의 [비밀번호 재설정] 버튼 클릭
     - 새 비밀번호(8자 이상) 입력 후 [변경]

  ▶ 유학원 비활성화:
     [비활성화] 버튼 클릭 시 로그인 불가 상태로 전환


--------------------------------------------------------------------------------
4-7. 리포트
--------------------------------------------------------------------------------

메뉴에서 [리포트]를 클릭합니다.

  - 상태별 학생 수 집계
  - 유학원별 학생 수 집계
  - 비자 만료 현황


================================================================================
5. 유학원 사용법
================================================================================

유학원 계정으로 로그인하면 본인 유학원 소속 학생만 볼 수 있습니다.

  ▶ 학생 목록 확인
  ▶ 학생 상세 정보 확인
  ▶ 학생 정보 수정 (이름, 연락처, 상태 등)

※ 유학원 계정으로는 학생 삭제 및 유학원 관리를 할 수 없습니다.
※ 계정 정보(이메일, 비밀번호) 변경은 관리자에게 문의하세요.


================================================================================
6. 학생 포털 사용법
================================================================================

접속 주소: https://aju-ej.vercel.app/portal

학생은 관리자로부터 받은 이메일/비밀번호로 로그인합니다.

확인 가능한 정보:
  - 본인 이름, 생년월일, 연락처
  - 현재 상태 (어학연수 / 대학교 / 취업 등)
  - 현재 소속 (어학원명 / 대학교명 / 회사명)
  - 비자 종류 및 만료일
  - 담당 유학원 정보

비밀번호 변경:
  상단 [계정] 탭 → [비밀번호 변경] → 현재 비밀번호와 새 비밀번호 입력 → [변경]

언어 변경:
  화면 우측 상단의 [한 | Vie] 버튼으로 한국어/베트남어 전환

탈퇴:
  [계정] 탭 → [탈퇴하기] → 확인 클릭 (탈퇴 후 로그인 불가)


================================================================================
7. 비자 만료 알림
================================================================================

비자 만료일이 등록된 학생에게 자동으로 이메일 알림이 발송됩니다.

발송 시점:
  - 만료 90일 전
  - 만료 30일 전
  - 만료 7일 전

※ 알림은 해당 날짜에 1회만 발송됩니다 (중복 발송 없음).
※ 학생 이메일이 등록되어 있어야 발송됩니다.
※ 이메일 내용은 한국어와 베트남어로 함께 작성되어 있습니다.

8. 자주 묻는 질문 (FAQ)
================================================================================

Q. 비밀번호를 잊었습니다.
A. 로그인 화면에서 [비밀번호 찾기]를 클릭하면 가입 이메일로 재설정 링크가
   발송됩니다. 유학원 계정의 경우 관리자에게 비밀번호 재설정을 요청하세요.

Q. 학생을 삭제할 수 있나요?
A. 완전 삭제는 지원하지 않습니다. [비활성화] 처리 시 목록에서 숨겨지며
   데이터는 보존됩니다.

Q. 여러 유학원 직원이 동시에 접속할 수 있나요?
A. 네, 가능합니다. 관리자가 같은 유학원에 여러 계정을 생성해 드리면
   각자 별도 계정으로 동시에 접속하여 작업할 수 있습니다.

Q. 모바일에서도 사용할 수 있나요?
A. 네, 스마트폰 브라우저에서도 사용 가능합니다.
   크롬(Chrome) 브라우저 사용을 권장합니다.

Q. 비자 알림 이메일이 오지 않습니다.
A. 다음을 확인해 주세요.
   1) 학생 정보에 이메일이 등록되어 있는지 확인
   2) 비자 만료일이 올바르게 입력되어 있는지 확인
   3) 스팸 메일함 확인

Q. Excel 내보내기 파일이 한글이 깨집니다.
A. Excel에서 파일을 열 때 인코딩을 UTF-8로 선택하거나,
   구글 스프레드시트에서 여는 것을 권장합니다.

Q. 인터넷이 없으면 사용할 수 없나요?
A. 이 시스템은 인터넷 연결이 필요한 웹 서비스입니다.
   오프라인에서는 사용할 수 없습니다.
