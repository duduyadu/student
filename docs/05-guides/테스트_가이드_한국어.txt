================================================================================
     AJU E&J 베트남 유학생 통합 관리 플랫폼 — 전체 테스트 가이드 (한국어)
     작성일: 2026년 3월
     대상: 처음 시스템을 사용하는 담당자
================================================================================

◆ 시스템 개요
--------------
이 시스템은 AJU E&J가 관리하는 베트남 유학생들의 정보를 통합 관리하는 웹 플랫폼입니다.
PC 브라우저(Chrome 권장)로 접속합니다.

- 시스템 주소: https://aju-ej.vercel.app
- 학생 포털 주소: https://aju-ej.vercel.app/portal

◆ 사용자 역할(Role) 안내
--------------------------
  ① master   : AJU E&J 총괄 관리자 — 전체 학생/유학원 관리 가능
  ② agency   : 유학원 담당자 — 본인 유학원 학생만 조회/관리 가능
  ③ student  : 학생 본인 — 포털에서 자신의 정보만 조회 가능

※ 테스트용 계정은 관리자(master)에게 별도로 받아주세요.

================================================================================
  PART 1 — MASTER(총괄 관리자) 기능 테스트
================================================================================

[TEST-M-01] 로그인
------------------
1. 브라우저에서 시스템 주소로 접속
2. 이메일 + 비밀번호 입력 후 '로그인' 클릭
3. 기대 결과: 대시보드 화면이 나타난다

[TEST-M-02] 대시보드 확인
--------------------------
1. 로그인 후 첫 화면(대시보드) 확인
2. 확인 항목:
   □ 상단 통계 카드: 전체 학생 수, 이번 달 신규, 비자 경보 수, 반려 서류 수 표시
   □ TOPIK 등급 분포 도넛 차트 (학생이 있을 경우 표시)
   □ 최근 활동 피드 (상담/신규 학생)
   □ 서류 현황 카드 (대기/제출/검토중/승인/반려 수)
   □ 비자 만료 경보 (D-7 이내: 빨간색, D-30 이내: 주황색, D-90 이내: 노란색)
   □ 상태별 학생 분포 (유학전/어학연수/대학교/취업)
3. 기대 결과: 모든 카드와 차트가 오류 없이 표시된다

[TEST-M-03] API 상태 모니터 확인 (master 전용)
------------------------------------------------
1. 대시보드 하단 'API 상태 모니터' 섹션 확인
2. '새로고침' 버튼 클릭
3. 기대 결과: Supabase DB, PDF API, AI API 등 상태가 초록(정상) 또는 빨간(오류)으로 표시

[TEST-M-04] 학생 목록 조회
---------------------------
1. 왼쪽 메뉴 또는 상단 '학생 목록' 클릭
2. 확인 항목:
   □ 학생 목록 표시 (사진 썸네일 포함)
   □ 검색창에 이름/코드 입력 시 필터링
   □ 상태 필터(유학전/어학연수/대학교/취업) 동작
3. 기대 결과: 학생 목록이 정상 표시된다

[TEST-M-05] 신규 학생 등록
---------------------------
1. '학생 목록' 페이지에서 '+ 신규 등록' 버튼 클릭
2. 아래 정보 입력:
   □ 한국어 이름 (예: 홍길동)
   □ 베트남어 이름 (예: Nguyen Van A)
   □ 생년월일
   □ 여권번호
   □ 전화번호(베트남)
   □ 소속 유학원 선택
   □ 상태: 유학전 선택
3. '저장' 버튼 클릭
4. 기대 결과: 저장 후 학생 상세 페이지로 이동, 학생코드 미배정 상태(승인 필요)

[TEST-M-06] 학생 승인 (대기 → 정식 등록)
------------------------------------------
1. 대시보드에서 '승인 대기 학생' 섹션 확인 (노란색 박스)
2. 방금 등록한 학생이 목록에 있는지 확인
3. '승인' 버튼 클릭
4. 기대 결과: 학생코드(예: 26001001) 자동 부여, 대기 목록에서 제거됨

[TEST-M-07] 학생 상세 조회 — 기본 정보
-----------------------------------------
1. 학생 목록에서 학생 이름 클릭
2. 확인 항목:
   □ 프로필 사진 영역 (클릭하면 사진 업로드 가능)
   □ 이름(한국어/베트남어), 생년월일, 여권번호, 상태, 비자 만료일
   □ 소속 유학원명
   □ 탭: 상담 / 시험 / 평가 / 서류 / 포털정보
3. 기대 결과: 모든 정보가 정상 표시된다

[TEST-M-08] 학생 정보 수정
---------------------------
1. 학생 상세 페이지에서 '수정' 버튼 클릭
2. 비자 만료일 변경 (테스트: 2027-12-31 입력)
3. '저장' 클릭
4. 기대 결과: 저장 완료 후 변경된 날짜가 반영됨

[TEST-M-09] 프로필 사진 업로드
-------------------------------
1. 학생 상세 페이지에서 프로필 사진 원(아바타) 클릭
2. JPG/PNG 파일 선택
3. 기대 결과: 사진이 업로드되고 썸네일로 표시됨

[TEST-M-10] 상담 기록 추가
---------------------------
1. 학생 상세 > '상담' 탭 클릭
2. '+ 상담 추가' 버튼 클릭
3. 상담 내용 입력 (예: "비자 연장 상담 완료")
4. '저장' 클릭
5. 기대 결과: 타임라인에 상담 기록이 추가됨

[TEST-M-11] TOPIK 시험 성적 추가
----------------------------------
1. 학생 상세 > '시험' 탭 클릭
2. '+ 성적 추가' 버튼 클릭
3. 아래 입력:
   □ 시험일: 2026-01-15
   □ 읽기 점수: 75
   □ 듣기 점수: 82
4. '저장' 클릭
5. 기대 결과:
   - 총점: 157점
   - 등급: 2급 (140점 이상)
   - 성적 차트에 반영됨

[TEST-M-12] AI 성적 분석
--------------------------
1. '시험' 탭 > 성적이 1개 이상 있는 상태에서 'AI 분석' 버튼 클릭
2. 잠시 대기 (Gemini AI 응답 시간)
3. 기대 결과: 학습 패턴 분석 및 개선 제안 텍스트가 표시됨

[TEST-M-13] 선생님 평가
------------------------
1. 학생 상세 > '평가' 탭 클릭
2. 4개 항목(출석률/학습태도/한국어실력/생활적응) 별점 입력
3. 의견 입력 (예: "성실하게 수업에 참여함")
4. '저장' 클릭
5. 기대 결과: 레이더 차트에 평가 내용 반영됨

[TEST-M-14] 비자 서류 체크리스트 관리
---------------------------------------
1. 학생 상세 > '서류' 탭 클릭
2. 서류 목록 확인 (여권사본, 재학증명서 등 10종)
3. 특정 서류에서 '승인' 버튼 클릭
4. 기대 결과:
   - 서류 상태가 '승인'으로 변경됨
   - 학생 이메일로 승인 알림 발송됨 (이메일 설정 시)

[TEST-M-15] 서류 반려 테스트
------------------------------
1. 다른 서류에서 '반려' 버튼 클릭
2. 반려 사유 입력 (예: "여권 사진이 흐릿합니다")
3. '확인' 클릭
4. 기대 결과:
   - 서류 상태가 '반려'로 변경됨
   - 학생 이메일로 반려 사유와 함께 알림 발송됨

[TEST-M-16] 생활기록부 PDF 다운로드 (단일)
--------------------------------------------
1. 학생 상세 > 우측 상단 '생활기록부 PDF' 버튼 클릭
2. 기대 결과:
   - 한국어(KO) PDF와 베트남어(VI) PDF 2개 파일 동시 다운로드
   - 학생 정보, 상담 기록, 성적, 평가 내용 포함

[TEST-M-17] 생활기록부 PDF 일괄 다운로드 (ZIP)
-------------------------------------------------
1. 학생 목록(/students) 페이지로 이동
2. 여러 학생의 체크박스 선택 (2명 이상)
3. '생활기록부 일괄 다운로드' 버튼 클릭
4. 기대 결과: ZIP 파일 1개 다운로드, 내부에 학생별 KO/VI PDF 포함

[TEST-M-18] Excel 학생 일괄 등록
----------------------------------
1. 학생 목록 > '일괄 등록' 버튼 클릭
2. Excel(.xlsx) 파일 업로드
   (샘플 파일 형식: 이름/생년월일/여권번호/유학원코드 열 포함)
3. '업로드' 클릭
4. 기대 결과: 파일의 학생들이 일괄 등록됨, 결과 리포트 표시

[TEST-M-19] 통계/리포트 페이지
--------------------------------
1. 메뉴에서 '리포트' 클릭
2. 확인 항목:
   □ 기간별 필터 (이번 달/지난 달/올해)
   □ 학생 수 통계
   □ 비자 만료 통계
   □ TOPIK 등급 분포
3. 기대 결과: 통계 수치가 정상 표시됨

[TEST-M-20] 유학원 관리
------------------------
1. 메뉴에서 '유학원' 클릭
2. 유학원 목록 확인
3. '+ 유학원 추가' 버튼 클릭
4. 유학원명(한국어/베트남어), 담당자 이메일 입력
5. '저장' 클릭
6. 기대 결과: 유학원 계정 생성됨, 이메일로 초대 메일 발송됨

[TEST-M-21] 언어 전환 (KO ↔ VI)
----------------------------------
1. 상단 우측의 [KO] 또는 [VI] 버튼 클릭
2. 기대 결과: 전체 UI 텍스트가 한국어 ↔ 베트남어로 전환됨

================================================================================
  PART 3 — STUDENT(학생 포털) 기능 테스트
================================================================================

[TEST-S-01] 학생 자가 등록
---------------------------
1. 브라우저에서 https://aju-ej.vercel.app/register 접속
2. 아래 입력:
   □ 이름(베트남어), 이름(한국어)
   □ 이메일 주소
   □ 비밀번호 (8자 이상)
   □ 생년월일, 여권번호
   □ 소속 유학원 선택
   □ 개인정보 동의 체크
3. '등록 신청' 클릭
4. 기대 결과:
   - "등록 신청이 완료되었습니다" 메시지 표시
   - 관리자(master) 승인 후 포털 이용 가능

[TEST-S-02] 학생 포털 로그인
------------------------------
1. https://aju-ej.vercel.app/portal 접속
2. 학생 이메일/비밀번호 입력
3. 기대 결과: 포털 메인 화면(내 정보) 표시

[TEST-S-03] 내 정보 조회
--------------------------
1. 포털 로그인 후 '내 정보' 탭 확인
2. 확인 항목:
   □ 이름, 학생코드, 소속 유학원
   □ 비자 종류 및 만료일
   □ 현재 상태 (유학전/어학연수/대학교/취업)
3. 기대 결과: 등록된 정보가 정상 표시됨

[TEST-S-04] 내 정보 수정
--------------------------
1. 포털 > '내 정보' > '수정' 버튼 클릭
2. 베트남 전화번호 수정 (예: 010-1234-5678)
3. '저장' 클릭
4. 기대 결과: 저장 완료 메시지 표시

[TEST-S-05] 서류 체크리스트 조회 및 업로드
--------------------------------------------
1. 포털 > '서류' 탭 클릭
2. 확인 항목:
   □ 필요 서류 목록 (10종) 및 현재 상태 표시
   □ '제출' 버튼이 있는 서류 확인
3. '제출' 버튼 클릭 → 메모 입력 후 '제출 완료' 클릭
4. 기대 결과: 서류 상태가 '대기' → '제출됨'으로 변경됨

[TEST-S-06] 상담 기록 조회
---------------------------
1. 포털 > '상담' 탭 클릭
2. 기대 결과: 담당자가 입력한 상담 내역 목록 표시 (수정 불가)

[TEST-S-07] 시험 성적 조회
---------------------------
1. 포털 > '성적' 탭 클릭
2. 기대 결과: TOPIK 성적 목록 및 등급 표시

[TEST-S-08] 비밀번호 변경
--------------------------
1. 포털 > '계정' 탭 클릭
2. '비밀번호 변경' 섹션에서 새 비밀번호 입력
3. '변경' 클릭
4. 기대 결과: "비밀번호가 변경되었습니다" 메시지 표시

[TEST-S-09] 언어 전환
-----------------------
1. 포털 상단의 [한국어/Tiếng Việt] 버튼 클릭
2. 기대 결과: 포털 전체 텍스트가 베트남어로 전환됨

[TEST-C-03] 로그아웃
---------------------

================================================================================
  PART 5 — 문제 발견 시 보고 방법
