# Plan: dashboard-enhancement

## 개요
메인 대시보드(`/`)를 개선하여 유학원별 학생 현황, 기간별 등록 추이, 서류 진행률 등 핵심 지표를 한눈에 파악할 수 있게 한다.

## 목표
- 유학원별 학생 수 비교 차트
- 월별/분기별 등록 추이 차트 (recharts 활용)
- 서류 승인률 현황 카드 개선
- TOPIK 등급 분포 통계
- 빠른 액션 버튼 (신규 학생 등록, 비자 만료 임박 목록)

## 현재 상태 분석
- `/` (대시보드): 기본 통계 카드 (전체 학생 수, 상태별 카운트)
- `/reports`: 상태별/유학원별/월별 통계 (bar chart)
- 대시보드와 리포트가 분리되어 중복 데이터 로딩

## 구현 범위

### P0 (필수)
1. **요약 카드 개선**
   - 전체 학생 / 이번 달 신규 / 비자 만료 임박(30일) / 서류 반려 대기
2. **TOPIK 등급 분포 도넛 차트** — 미취득/1급/2급 비율 (recharts)
3. **최근 활동 피드** — 최근 상담·서류 승인·신규 등록 타임라인

### P1 (권장)
4. **유학원별 학생 현황 바 차트** (리포트 페이지에서 대시보드로 이관)
5. **빠른 액션**: 신규 학생 등록, 비자 만료 임박 학생 보기
6. **역할별 대시보드** — master와 agency가 보는 화면 다르게

## 관련 파일
- `app/page.tsx` (대시보드 메인)
- `app/reports/page.tsx` (통계 — 일부 로직 공유)
- `components/ExamChart.tsx` (recharts 패턴 참고)
- `lib/supabase.ts`
